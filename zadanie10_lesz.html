Budowa selektora w CSS

1. Główne części selektora

Każdy selektor składa się z dwóch podstawowych elementów:

  • Selektor – określa, do jakiego elementu HTML będą stosowane style (np. h1).
  • Deklaracja – zawiera zestaw właściwości i wartości, np. color: blue;.

2. Znaczenie nawiasów klamrowych { ... }

Nawiasy klamrowe otaczają zbiór deklaracji przypisanych do danego selektora. Każda deklaracja definiuje konkretny wygląd elementu.

3. Opis cech w CSS

Cechy w CSS to właściwości stylu, które określają wygląd elementów. Przykłady:

  • color – kolor tekstu
  • font-size – rozmiar czcionki
  • margin – marginesy wokół elementu

4. Opis wartości w CSS

Wartości w CSS są przypisane do cech i definiują konkretny wygląd elementu. Przykłady:

  • Dla koloru: red, #FFFF00
  • Dla rozmiaru: 12px, 1.5em

5. Kiedy stosować średnik ;?

Średnik oddziela poszczególne deklaracje wewnątrz selektora:

Ostatnia deklaracja może, ale nie musi mieć średnika. Jednak jego stosowanie na końcu zwiększa przejrzystość kodu.
