Identyfikatory — ID

Pojęcie Opis
Co znaczy skrót ID? ID to skrót od „identyfikator” – unikalna nazwa przypisana do elementu HTML.
Jaki jest cel stosowania ID? Stosujemy ID, aby jednoznacznie zidentyfikować element na stronie i przypisać mu indywidualny styl CSS.
Jakie jest znaczenie znaku #? Znak # w CSS służy do odwoływania się do elementu z określonym ID.
Jak następuje wywołanie ID (wraz z wytłumaczeniem)? Wywołanie ID w CSS: np. #naglowek { color: red; } – selektor # odnosi się do elementu z id="naglowek"
Uwagi na temat stosowania ID
  • ID musi być unikalne na stronie.
  • W przypadku wielu podobnych elementów lepiej użyć klasy class.
  • Użycie wielu elementów z tym samym ID nie jest błędem krytycznym, ale niezalecane.
