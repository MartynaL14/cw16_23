1. Opis znacznika DIV

DIV to element blokowy służący do grupowania innych elementów HTML. Składnia: <div>...</div>. Stosowany jest do tworzenia sekcji i układów. Obsługuje m.in. atrybut ALIGN, który określa wyrównanie zawartości.

2. Opis znacznika SPAN

SPAN to element liniowy, służący do grupowania tekstu w linii. Składnia: <span>...</span>. Używany do stylowania fragmentów tekstu.

3. Różnice między znacznikami DIV i SPAN

DIV to element blokowy – zajmuje całą szerokość. SPAN to element liniowy – zajmuje tylko tyle miejsca, ile potrzebuje jego zawartość.

4. Tagi

Blokowe: div, p, h1–h6, ul, ol, li, table
Liniowe: span, a, b, i, u, img, input
Liniowo-blokowe: button, textarea
